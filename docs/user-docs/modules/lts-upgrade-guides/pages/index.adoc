= Jenkins LTS Upgrade Guide

This section highlights important changes for administrators upgrading link:https://www.jenkins.io/download/lts/[Jenkins LTS].
Each section covers the upgrade from the previous LTS release, sections on versions x.y.1 cover the upgrade from the last release of the previous LTS line.
if you are skipping LTS releases when upgrading, it is recommended to read the sections for all releases in between.
