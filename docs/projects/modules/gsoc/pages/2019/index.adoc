= Jenkins in Google Summer of Code 2019

Jenkins project participated in Google Summer of Code 2019.
This page contains archive of the 2019-specific information.
See the xref:projects:gsoc:index.adoc[main GSoC page] for the actual information.

Completed projects:

* xref:projects:gsoc:2019/remoting-over-apache-kafka-docker-k8s-features.adoc[Remoting over Apache Kafka with Kubernetes features]
* xref:projects:gsoc:2019/role-strategy-performance.adoc[Role Strategy Plugin Performance Improvement]
* xref:projects:gsoc:2019/working-hours-improvements.adoc[Working Hours Plugin - UI Improvements]
